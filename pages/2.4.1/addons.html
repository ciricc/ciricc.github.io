Дополнения к Easy VK

В этом разделе я собираю все дополнения, которые были сделаны стороннимм разработчиками и мною специально для Easy VK

vk-bots

Надстройка для создания чат-ботов с помощью Easy VK от разработчика Easy VK

Github:

github.com/ciricc/vk-bots

storybot

Бот, который будет смотреть истории пользователей ВКонтакте (масслукинг для ВК). Точнее его API для запуска и отладки

Github:

github.com/ciricc/storybot

simple-bots

Надстройка для создания чат-ботов с помощью Easy VK с простым и доступным интерфейсом

Github:

github.com/khok/simple-bots
