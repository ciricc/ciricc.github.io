Виджеты

Виджеты в Easy VK - это инструменты, которые неофициально позволяют легко и просто делать что-то такое, чего нет в официальном API ВКонтакте. Это не HTTP клиент, так как не требует авторизации вообще в принципе, это просто удобный инструмент и набор любителя

# widgets.getLiveViews()

Метод возвращает количество просмотров Live трансляции

Аргументы

  • source_video * (string) - Сорц на видеозапись, полный идентификатор по сути видеозаписи из URL адреса и не только
    [+] (Спойлер) - Пример live_source (изображение) 



vk.widgets.getLiveViews('-157276298_456239074').then(views => {
	console.log(views)   
})
