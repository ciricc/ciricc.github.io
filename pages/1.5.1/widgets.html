Виджеты

Виджеты - это инструменты, которые помогают делать что-то неофициально, то, что не поддерживается в ВКонтакте API, как например, получить количество просмотров Live Видео

Обычно обращение к виджетам происходит таким образом 

vk.widgets.exampleWidget()

widgets.getLiveViews(videoSource)

Данный метод позволяет получить количество просмотров Live видео. Я заметил, что кому-то на форуме по ВКонтакте API понадобился скрипт получения просмотров, мне стал интересен этот вопрос и я решил, что это будет полезной фитчей в EasyVK.

Для того, чтобы получить количество просмотров, вам необходимо первым аргументом отправит sourceId видео, как со скриншота.

Затем использовать виджет 

easyvk(/*параметры авторизации*/).then(vk => {
vk.widgets.getLiveViews('-157276298_456239074').then(views => {
console.log(views)   
})
})
